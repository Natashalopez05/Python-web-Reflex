## Pasos:
* mkdir link_bio
* cd link_bio
* python3 -m venv venv (crear un entorno virtual)
* source venv/bin/activate (para activar el entorno virtual)

## Instalar reflex y ejecutarlo:
* pip install reflex
* reflex init (para iniciar)
* reflex run (para correr el proyecto)
